# Security Policy

## Reporting a vulnerability

Report security issues privately via GitHub Security Advisories for [xgic/gitlab-graphql](https://github.com/xgic/gitlab-graphql) when available, or contact maintainers via [github.com/xgic](https://github.com/xgic).

Do **not** open a public issue with exploit details, tokens, or secrets.

## Public-safe contributions

Do not include private infrastructure details, private tracker IDs, internal paths, or credentials in issues, PRs, or commits.

Multi-repo standards: https://github.com/xgic/ai
